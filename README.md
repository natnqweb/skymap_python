# skymap_python

### example

    python skymap.py calculate-all [latitude] [longitude] [Dec] [Ra] [year] [month] [day] [utc]
    los angeles and sirius example:
    python skymap.py calculate-all 34.052235 -118.243683 -16.7424 101.52 2021 9 4 20
